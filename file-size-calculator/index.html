容量計算ツール

テキストをファイルとして保存した場合のサイズを計算します

  • ホームに戻る

テキスト入力

計算結果

テキストを入力して「サイズを計算」ボタンをクリックしてください。

使用方法

  1. 上のテキストエリアに計算したいテキストを入力します
  2. 「サイズを計算」ボタンをクリックします
  3. 結果エリアにファイルサイズが表示されます
  4. 「テキストをクリア」ボタンで入力内容を削除できます

特徴

  • UTF-8エンコーディングでのバイト数を正確に計算
  • バイト、KB、MB単位で表示
  • 文字数、行数も同時に表示
  • リアルタイムで計算結果を更新
