URLエンコーディングについて

URLエンコーディングは、URL内で安全に使用するために特殊文字を変換する仕組みです。

予約文字

URL内で特別な意味を持つ文字

! * ' ( ) ; : @ & = + $ , / ? # [ ]

非安全文字

必ずエンコードが必要な文字

スペース " < > # % { } | \ ^ ~ ` [ ]

エンコード例

よく使われる文字の変換例

スペース → %20 または +
日本語 → %E3%81%82 (あ)
