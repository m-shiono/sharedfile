使用方法

  1. Excelの表の必要な個所を選択し、コピーします。
  2. テキストエリアにカーソルを置き、貼り付けます。
  3. 出力形式を選択し、「変換」ボタンをクリックします。
  4. 下のテキストエリアに選択した形式のコードが表示されます。

注意事項

このシステムは単純な表データに対してのみ機能し、Excelの書式設定やセルの結合に対応していません。
