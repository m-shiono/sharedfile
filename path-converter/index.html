Path変換パターン

このツールは以下のパス形式間での相互変換を行います。

Windows形式

バックスラッシュ区切り

C:\test\hoge.txt Windows標準のパス形式

Unix形式

スラッシュ区切り

/home/user/test.txt Linux/macOS標準のパス形式

エスケープ済み

プログラミング用

C:\\test\\hoge.txt JavaScript、JSON等で使用

URL形式

Web用パス

file:///C:/test/hoge.txt ブラウザ、Web API用

UNC形式

ネットワークパス

\\server\share\file Windows共有フォルダ

POSIX形式

標準Unix形式

/mnt/share/file.txt POSIX準拠システム

使用方法

  1. 自動変換モード: パスを入力すると自動的に判定し、全形式に変換
  2. 手動変換モード: 変換元と変換先の形式を指定して変換
  3. サンプルボタン: 各種パス形式のサンプルを確認
  4. 結果コピー: 各変換結果をクリックでクリップボードにコピー
