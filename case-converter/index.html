入力テキスト

変換結果

camelCase 変数名・関数名（JavaScript、Java等）
PascalCase クラス名・型名（C#、Pascal等）
snake_case 変数名・関数名（Python、Ruby等）
SCREAMING_SNAKE_CASE 定数名（多くの言語）
kebab-case URL・CSS クラス名・HTML属性
SCREAMING-KEBAB-CASE 環境変数・設定項目
Train-Case ヘッダー名・タイトル
dot.case 設定ファイル・オブジェクトパス
path/case ファイルパス・URL パス
Title Case 見出し・タイトル
Sentence case 通常の文章
lowercase 全て小文字
UPPERCASE 全て大文字

クイック例文

命名規則について

JavaScript/TypeScript

  • 変数・関数: camelCase
  • クラス・型: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • ファイル名: kebab-case または camelCase

Python

  • 変数・関数: snake_case
  • クラス: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • モジュール: snake_case

Java/C#

  • 変数・メソッド: camelCase
  • クラス・インターフェース: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • パッケージ: lowercase

Web開発

  • CSS クラス: kebab-case
  • HTML 属性: kebab-case
  • URL: kebab-case
  • 環境変数: SCREAMING_SNAKE_CASE
