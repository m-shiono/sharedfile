命名規則について

JavaScript/TypeScript

  • 変数・関数: camelCase
  • クラス・型: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • ファイル名: kebab-case または camelCase

Python

  • 変数・関数: snake_case
  • クラス: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • モジュール: snake_case

Java/C#

  • 変数・メソッド: camelCase
  • クラス・インターフェース: PascalCase
  • 定数: SCREAMING_SNAKE_CASE
  • パッケージ: lowercase

Web開発

  • CSS クラス: kebab-case
  • HTML 属性: kebab-case
  • URL: kebab-case
  • 環境変数: SCREAMING_SNAKE_CASE
